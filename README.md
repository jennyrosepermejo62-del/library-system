# 📚 Library System

A responsive library management system built with **PHP** and **MySQL**. It features a clean UI and allows users to manage book records efficiently.

### 🚀 Features
* **Full CRUD:** Add, View, Edit, and Delete books.
* **Authentication:** Secure user login system.
* **Responsive:** Works on mobile and desktop.

### 🛠️ Tech Stack
* **Language:** PHP
* **Database:** MySQL
* **Frontend:** HTML, CSS, JavaScript

### ⚙️ Setup
1. Import `database.sql` into your PHPMyAdmin.
2. Place the folder in your `htdocs` directory.
3. Open `localhost/library-system` in your browser.
